# Accessibility Settings

CleverKeys includes accessibility features for users with visual, motor, or cognitive accessibility needs.

## Quick Summary

| What | Description |
|------|-------------|
| **Purpose** | Accessibility accommodations |
| **Access** | Settings > Accessibility |
| **Features** | High contrast, large keys, TalkBack |

## Visual Accessibility

### High Contrast Mode

Enhanced visual contrast:

| Setting | Effect |
|---------|--------|
| **Enabled** | Bold borders, increased contrast |
| **Disabled** | Standard appearance |

### Font Size

Adjust key label size:

| Size | Use Case |
|------|----------|
| **Small** | More keys visible |
| **Normal** | Default |
| **Large** | Easier reading |
| **Extra Large** | Low vision |

### Key Label Style

| Style | Description |
|-------|-------------|
| **Standard** | Regular labels |
| **Bold** | Thicker font weight |
| **Uppercase** | All caps labels |

## Screen Reader Support

### TalkBack Integration

CleverKeys is compatible with Android TalkBack:

| Feature | Behavior |
|---------|----------|
| **Key Announcement** | Speaks key on focus |
| **Input Feedback** | Announces typed text |
| **Navigation** | Supports explore by touch |

### Using with TalkBack

1. Enable TalkBack in Android settings
2. CleverKeys automatically adapts
3. Explore keyboard by touch
4. Double-tap to type

### Spoken Feedback

| Setting | Description |
|---------|-------------|
| **Speak Keys** | Announce key on press |
| **Speak Words** | Announce completed words |
| **Speak Predictions** | Announce suggestions |

## Motor Accessibility

### Touch Accommodation

For users with motor difficulties:

| Setting | Description |
|---------|-------------|
| **Touch Duration** | Required hold time |
| **Ignore Repeat** | Prevent accidental repeats |
| **Touch Slop** | Movement tolerance |

### Large Touch Targets

| Setting | Effect |
|---------|--------|
| **Increased Padding** | Bigger tap areas |
| **Reduced Keys** | Fewer, larger keys |
| **Extra Large Height** | Maximum keyboard height |

### Sticky Modifiers

For users who can't hold two keys:

| Setting | Description |
|---------|-------------|
| **Enabled** | Tap shift, then letter |
| **Disabled** | Must hold shift |

## Timing Settings

### Long Press Delay

Adjust time for long press:

| Duration | Use Case |
|----------|----------|
| **Short (200ms)** | Quick access |
| **Default (400ms)** | Standard |
| **Long (800ms)** | Prevent accidental |
| **Extra Long (1200ms)** | Significant delay |

### Key Repeat Delay

If holding a key:

| Setting | Description |
|---------|-------------|
| **Slow** | Longer before repeat |
| **Normal** | Standard repeat |
| **Fast** | Quick repeat |
| **Disabled** | No repeat on hold |

## Cognitive Accessibility

### Simplified Mode

Reduce complexity:

| Feature | Simplified |
|---------|------------|
| **Gestures** | Disabled |
| **Subkeys** | Disabled |
| **Predictions** | Single row |
| **Layout** | Basic QWERTY only |

### Consistent Layout

| Setting | Description |
|---------|-------------|
| **Lock Layout** | Prevent accidental switches |
| **Hide Globe** | Remove layout switcher |

## Tips and Tricks

- **Low vision**: Enable high contrast + large font
- **Motor difficulties**: Increase touch duration, enable sticky keys
- **Screen reader**: Use TalkBack with spoken feedback
- **Cognitive**: Try simplified mode

> [!TIP]
> Android's accessibility settings work alongside CleverKeys. Enable both for best results.

## Common Questions

### Q: How do I use CleverKeys with TalkBack?

A: Enable TalkBack in Android settings. CleverKeys automatically provides spoken feedback and explore-by-touch navigation.

### Q: Can I make just the letters bigger?

A: Yes, adjust Font Size in Accessibility settings. This affects key labels without changing keyboard size.

### Q: How do I prevent accidental key presses?

A: Increase Touch Duration to require holding keys longer, or enable Simplified Mode.
