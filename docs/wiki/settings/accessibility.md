# Accessibility Settings

CleverKeys includes haptic feedback and audio settings in the Accessibility section.

## Quick Summary

| What | Description |
|------|-------------|
| **Purpose** | Haptic feedback and sound settings |
| **Access** | Settings > Accessibility |
| **Features** | Vibration, haptic triggers, keypress sound |

## Vibration Settings

### Master Vibration Toggle

Enable/disable all haptic feedback:

| Setting | Effect |
|---------|--------|
| **Enabled** | Vibration feedback active |
| **Disabled** | No vibration feedback |

### Vibration Duration

Custom vibration length in milliseconds:

| Range | Description |
|-------|-------------|
| **1-100ms** | Adjust duration of haptic feedback |
| **Default: 20ms** | Brief, subtle feedback |

## Haptic Feedback Triggers

Configure which actions trigger haptic feedback:

| Trigger | Description |
|---------|-------------|
| **Key Press** | Vibrate on each key tap |
| **Suggestion Tap** | Vibrate when selecting a prediction |
| **TrackPoint** | Vibrate when entering trackpoint mode |
| **Long Press** | Vibrate on long press actions |
| **Swipe Complete** | Vibrate when swipe word is inserted |

Each trigger can be independently enabled/disabled.

## Sound Settings

### Sound on Keypress

Play audio feedback on key taps:

| Setting | Effect |
|---------|--------|
| **Enabled** | Click sound on each key |
| **Disabled** | Silent typing |

## Tips

- **Battery conscious**: Disable vibration to save battery
- **Silent typing**: Disable both vibration and sound
- **Subtle feedback**: Use short vibration duration (10-15ms)
- **Strong feedback**: Use longer vibration duration (40-50ms)

## System Integration

CleverKeys respects Android's system-wide accessibility settings where applicable. For additional accessibility features like TalkBack screen reader support, configure them in Android's system accessibility settings.
