# Reset to Defaults

Restore CleverKeys settings to their original state when troubleshooting issues or wanting a fresh start.

## Quick Summary

| Reset Type | What's Affected |
|------------|-----------------|
| **Settings Only** | Preferences, keeps dictionary |
| **Customizations** | Per-key actions, extra keys |
| **Full Reset** | Everything except app installation |

## Reset Options

### Option 1: Reset Settings Only

Restores preference settings to defaults:

| Affected | Not Affected |
|----------|--------------|
| Appearance settings | Personal dictionary |
| Input behavior | Clipboard history |
| Haptics | Profiles |
| Gestures | Customizations |

**How to:**

1. Open **Settings**
2. Scroll to **Reset**
3. Tap **Reset Settings**
4. Confirm

### Option 2: Reset Customizations

Restores per-key customizations:

| Affected | Not Affected |
|----------|--------------|
| Per-key actions | Settings |
| Extra keys setup | Dictionary |
| Subkey customizations | Profiles |

**How to:**

1. Open **Settings**
2. Go to **Customization**
3. Tap **Reset All Customizations**
4. Confirm

### Option 3: Reset Layouts

Restores layout configuration:

| Affected | Not Affected |
|----------|--------------|
| Installed layouts | Settings |
| Layout order | Dictionary |
| Custom layouts | Customizations |

**How to:**

1. Open **Settings > Layouts**
2. Tap **Reset Layouts**
3. Confirm

### Option 4: Clear Dictionary

Removes learned words:

| Affected | Not Affected |
|----------|--------------|
| Personal dictionary | Settings |
| Learned patterns | Layouts |
| Word frequency | Customizations |

**How to:**

1. Open **Settings > Privacy**
2. Tap **Clear Personal Dictionary**
3. Confirm

### Option 5: Full Reset

Restores everything to factory state:

| Affected |
|----------|
| All settings |
| All customizations |
| Personal dictionary |
| Clipboard history |
| Profiles |
| Learned patterns |

**How to:**

1. Open **Settings**
2. Scroll to **Reset**
3. Tap **Reset Everything**
4. Type "RESET" to confirm
5. Keyboard returns to factory state

> [!WARNING]
> Full reset cannot be undone. Export your profile first if you want to restore later.

## Before Resetting

### Backup First

Before any reset, consider backing up:

1. **Export Profile**
   - Settings > Profiles > Export
   - Saves all settings and customizations

2. **Export Dictionary**
   - Settings > Privacy > Export Dictionary
   - Saves personal words

See [Backup & Restore](backup-restore.md) for details.

### Try Less Drastic Options

Before full reset:

1. **Clear app cache** (doesn't delete data)
   - Android Settings > Apps > CleverKeys > Storage > Clear Cache

2. **Reset single category** instead of everything

3. **Force stop** the app
   - Android Settings > Apps > CleverKeys > Force Stop

## After Resetting

### First-Time Setup

After reset, go through setup:

1. **Enable keyboard** in system settings if needed
2. **Set as default** keyboard
3. **Choose layout** preference
4. **Configure settings** as desired

### Restore from Backup

If you exported a profile:

1. Go to **Settings > Profiles**
2. Tap **Import Profile**
3. Select your backup file
4. Settings are restored

## Reset for Specific Issues

| Issue | Recommended Reset |
|-------|-------------------|
| **Wrong corrections** | Clear Dictionary |
| **Gesture issues** | Reset Settings |
| **Layout problems** | Reset Layouts |
| **Visual glitches** | Reset Settings |
| **General weirdness** | Full Reset |

## Individual Setting Reset

For single settings:

1. Go to the setting category
2. Long-press the specific setting
3. Select **Reset to Default**

Or:

1. Open setting
2. Look for **Default** button
3. Tap to reset that setting only

## Common Questions

### Q: Will reset delete my downloaded language packs?

A: No, language packs are preserved. Only personalization is reset.

### Q: Can I undo a reset?

A: No, but you can restore from a profile backup if you made one.

### Q: Will reset fix my keyboard not appearing?

A: Maybe. If the issue is configuration, yes. If it's a system issue, you may also need to re-enable in Android settings.

### Q: Does uninstalling and reinstalling do the same thing?

A: Yes, but reinstalling is slower and may require you to re-enable the keyboard in system settings.
