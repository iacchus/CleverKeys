Clipboard Shortcuts

Use keyboard shortcuts for fast copy, cut, paste, and other clipboard operations.

Quick Summary

WhatDescription

PurposeFast clipboard operations

Standard Shortcuts

With Ctrl Key

Add Ctrl as an extra key to use standard shortcuts:

ShortcutAction

Ctrl+CCopy selected text
Ctrl+XCut selected text
Ctrl+VPaste
Ctrl+ASelect all
Ctrl+ZUndo
Ctrl+YRedo

Setting Up Ctrl Key

  • 1. Go to Settings > Activities > Extra Keys
  • 2. Add Ctrl to left or right side
  • 3. Ctrl key appears on bottom row
  • 4. Tap Ctrl, then the letter key

Gesture Shortcuts

Circle Gestures

Draw circles for undo/redo:

GestureAction

Counter-clockwise circleUndo
Clockwise circleRedo

See Circle Gestures for details.

Ctrl Key Subkeys

Swipe from Ctrl key for quick access:

DirectionAction

Swipe Up (N)Select All (Ctrl+A)
Swipe Right (E)Copy (Ctrl+C)
Swipe Down (S)Paste (Ctrl+V)
Swipe Left (W)Cut (Ctrl+X)

Paste from Clipboard

To paste the most recently copied text:

  • • Use Ctrl+V (with Ctrl extra key)
  • • Or swipe S on Ctrl key (Ctrl→S subkey)

To paste from clipboard history, open the clipboard panel (Ctrl SW swipe) and tap an item.

Customizing Shortcuts

Per-Key Actions

Assign clipboard actions to swipe directions:

  • 1. Go to Settings > Activities > Per-Key Customization
  • 2. Select a key (e.g., Shift)
  • 3. Choose a direction
  • 4. Select action (Copy, Paste, etc.)

Example setup:

KeyDirectionAction

ShiftNECopy
ShiftSEPaste
ShiftSWCut

See Per-Key Actions.

Tips and Tricks

Common Questions

Q: Why isn't Ctrl+C working?

A: First tap Ctrl, then tap C. They're not simultaneous presses.

Q: Can I hold Ctrl like on a computer?

A: The modifier is "sticky" - tap once, it applies to next key. No need to hold.

Q: How do I make shortcuts faster?

A: Use Ctrl subkeys (swipe from Ctrl) instead of two taps.
