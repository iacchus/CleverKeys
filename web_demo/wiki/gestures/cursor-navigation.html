Cursor Navigation

Move the text cursor precisely using the spacebar slider or TrackPoint mode without lifting your fingers from the keyboard.

Quick Summary

WhatDescription

PurposePosition cursor in text without tapping screen
MethodsSpacebar swipe, TrackPoint mode
PrecisionProportional to swipe speed/distance

Method 1: Spacebar Swipe

The most intuitive way to move the cursor:

How to Use

  • 1. Touch the spacebar and swipe left or right
  • 2. Cursor moves proportionally to swipe distance
  • 3. Release when positioned

Features

  • • Proportional speed: Faster/longer swipe = cursor moves further
  • • Visual feedback: Trail shows movement direction
  • • Immediate: Cursor moves as you swipe
[!TIP]
Slow, short swipes give fine control for precise positioning.

Method 2: TrackPoint Mode

For continuous joystick-style control:

How to Use

  • 1. Long-press the nav key (between spacebar and enter)
  • 2. Feel haptic activation (if enabled)
  • 3. Move finger in any direction like a joystick
  • 4. Cursor moves continuously in that direction
  • 5. Release to stop

See TrackPoint Mode for full details.

Navigation Summary

GestureAction

Swipe spacebar leftMove cursor left
Swipe spacebar rightMove cursor right
Long-press nav keyEnter TrackPoint mode

Tips and Tricks

  • • Precision: Tap text field to position roughly, then fine-tune with spacebar
  • • Speed: TrackPoint mode is good for large movements
  • • Accuracy: Slow spacebar swipes are most precise for small adjustments
  • • Muscle memory: Practice spacebar sliding - it becomes natural quickly
[!NOTE]
Cursor navigation works in all text fields, but some apps may handle input differently.

Common Questions

Q: Why doesn't spacebar slider work?

A: Make sure you're swiping horizontally on the spacebar. Very short swipes may be interpreted as taps.

Q: How do I select text while moving cursor?

A: Use Selection Delete mode by short-swiping on the backspace key and holding.

Q: Can I move cursor vertically?

A: Use TrackPoint mode for vertical cursor movement by long-pressing the nav key.
