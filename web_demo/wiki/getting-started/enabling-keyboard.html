Enabling the Keyboard

After installing CleverKeys, you need to enable it in Android settings and set it as your default keyboard.

Quick Summary

WhatDescription
PurposeEnable CleverKeys as your input method
AccessSettings > System > Keyboard
Time1-2 minutes

How to Enable

Step 1: Open Language & Input Settings

  • 1. Open Settings on your device
  • 2. Tap System (or search for "keyboard")
  • 3. Tap Languages & input
  • 4. Tap On-screen keyboard or Virtual keyboard

Note

Menu names vary by device manufacturer. Look for "keyboard" or "input" settings.

Step 2: Enable CleverKeys

  • 1. Tap Manage keyboards or the + button
  • 2. Find CleverKeys in the list
  • 3. Toggle it ON
  • 4. Accept the security warning (all keyboards show this)

Step 3: Set as Default

  • 1. Tap Default keyboard or Current keyboard
  • 2. Select CleverKeys

Alternatively, when typing in any app:

  • 1. Tap the keyboard icon in the navigation bar
  • 2. Select CleverKeys

Quick Switch Method

Once enabled, you can quickly switch keyboards:

  • • Navigation bar: Tap the keyboard icon (usually bottom-right)
  • • Notification: Swipe down and tap "Change keyboard"
  • • Long-press space: In some apps, long-press spacebar to switch

Tips and Tricks

  • • First launch: CleverKeys will guide you through initial setup
  • • Permissions: Grant notification access for clipboard features
  • • Default keyboard: Make CleverKeys default to avoid switching each time

Troubleshooting

CleverKeys not appearing in keyboard list

  • • Restart your device after installation
  • • Reinstall the app

Keyboard won't show up when typing

  • • Ensure CleverKeys is enabled in keyboard settings
  • • Tap the keyboard icon to manually select it

Security warning about keyboard

  • • This is normal for all third-party keyboards
  • • CleverKeys is open source and doesn't transmit your typing data
