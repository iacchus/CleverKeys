Enabling the Keyboard

Android keyboard activation steps

Enabling the Keyboard

After installing CleverKeys, you need to enable it in Android Settings and set it as your default keyboard.

Quick Steps

  • 1. Go to Settings → System → Languages & Input → On-screen keyboard
  • 2. Tap Manage on-screen keyboards
  • 3. Enable CleverKeys
  • 4. Go back and tap Default keyboard
  • 5. Select CleverKeys

Detailed Instructions

Step 1: Open Settings

Open the Android Settings app (gear icon in your app drawer or notification shade).

Step 2: Find Keyboard Settings

Navigate to: System → Languages & Input → On-screen keyboard

Note: The exact path varies by Android version and manufacturer. You can also search for "keyboard" in Settings.

Step 3: Enable CleverKeys

  • 1. Tap Manage on-screen keyboards
  • 2. Find CleverKeys in the list
  • 3. Toggle it ON
  • 4. Accept the warning about input method security

Step 4: Set as Default

  • 1. Go back to On-screen keyboard
  • 2. Tap Default keyboard (or similar)
  • 3. Select CleverKeys

Alternative: Quick Switch

You can also switch keyboards when any text field is active:

  • 1. Tap any text input field
  • 2. Look for the keyboard icon in your navigation bar
  • 3. Tap it and select CleverKeys

Verification

To verify CleverKeys is working:

  • 1. Open any app with a text field (Notes, Messages, etc.)
  • 2. Tap the text field
  • 3. You should see the CleverKeys keyboard appear

Troubleshooting

Keyboard doesn't appear

  • • Restart your device
  • • Make sure CleverKeys is enabled AND set as default
  • • Check if another keyboard is overriding it

Can't find keyboard settings

Search for "keyboard" or "input" in your Settings app. Different manufacturers place this in different locations.
