Special Characters

Access symbols, accented letters, diacritics, and currency symbols through long-press menus and short swipe gestures.

Quick Summary

WhatDescription
PurposeType characters not on main keyboard
AccessLong-press or short swipe on keys
TypesSymbols, accents, currency, punctuation

Methods to Access Special Characters

Method 1: Long-Press

Hold any key to see available special characters:

  • 1. Touch and hold a key for ~400ms
  • 2. A popup appears showing available characters
  • 3. Slide to select the character you want
  • 4. Release to type it

Common long-press characters:

Method 2: Short Swipe

Faster access via quick flick gestures:

  • 1. Touch the key
  • 2. Flick in the direction of the subkey
  • 3. The character types immediately

Example on 'e' key:

  • • Swipe North: Number (3)
  • • Swipe South: é (common accent)
  • • Swipe East/West: Other variations

Method 3: Symbol Keyboard

For extensive symbols:

  • 1. Tap ?123 to switch to numbers/symbols
  • 2. Tap =\< for more symbols
  • 3. Find mathematical, currency, and special symbols

Accented Characters (Diacritics)

For languages requiring diacritics:

Acute Accent (é)

  • • Long-press the base letter
  • • Select from popup

Grave Accent (è)

  • • Long-press the base letter
  • • Select from popup

Circumflex (ê)

  • • Long-press the base letter
  • • Select from popup

Umlaut/Diaeresis (ë)

  • • Long-press the base letter
  • • Select from popup

Tilde (ñ)

  • • Long-press n or a/o
  • • Select from popup

Cedilla (ç)

  • • Long-press c
  • • Select from popup

Tips and Tricks

  • • Learn subkeys: Check which characters are in which direction
  • • Speed: Short swipes are faster than long-press once learned
  • • Customize: You can change subkey assignments in Settings
  • • Language packs: Some accents only appear with language pack installed

Tip

If you frequently use certain special characters, consider customizing subkey positions to place them in easier-to-reach directions.
